# 📩 Spam Email Classifier

This project uses Machine Learning (Naive Bayes + TF-IDF) to classify messages as Spam or Not Spam.

## 🚀 Features
- Detect spam messages
- Shows prediction probability
- Simple web interface using Streamlit

## 🧠 Tech Stack
- Python
- Scikit-learn
- NLP (TF-IDF)
- Streamlit

## 📂 Dataset
SMS Spam Collection Dataset

## ▶️ Run Locally
```bash
pip install -r requirements.txt
python model.py
streamlit run app.py
